# SimpleWeather

A weather app for Android.

Introduction

  • Get weather for current location

  • Manually add cities by name or zip code

  • Get 5 day detailed forecast

  • Recieve weather notification on a per city basis

Uses:

  • OpenWeatherMap

  • RxJava2

  • RxAndroid

  • Android-Job

Pre-requisites

  • Minimum Android SDK 19
